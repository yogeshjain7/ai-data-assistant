# AI-Powered Intelligent Data Assistant

An AI-driven conversational analytics application built using Streamlit, Pandas, Matplotlib, and Google Gemini API.

## Overview
This project allows users to upload structured CSV datasets and interact with them using natural language queries. The system automatically performs statistical profiling, detects data quality issues, generates visualizations, and produces AI-powered business insights.

## Key Features
- CSV Upload & Automatic Analysis
- Statistical Profiling (Missing Values, Duplicates, Summary Stats)
- Dynamic Visualizations
- AI-Generated Business Insights
- Natural Language Query Interface

## Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib
- Google Gemini API

## How to Run
pip install -r requirements.txt  
streamlit run app.py
